# OPEX Tracker

Console-based Java application

## Project goal

The goal of this project is to implement a simple financial management system focused on fixed operational costs such as rent, utilities, telecommunications, security, cleaning.

## Features (MVP)

- Add operational expenses
- Categorize expenses by vendor and service type
- Separate fixed and variable costs
- Generate basic expense reports

## Technologies

- Java 24
- Maven
- Git
- IntelliJ IDEA
